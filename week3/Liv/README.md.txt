### Description
This project is part of my C# week3 training. The app will find 2 digit in an array that result is the target sum and return the index.

## How to run the project
A) CLI
1. Clone this repository
2. Open command prompt and go the this path "...\week3\Liv\TwoSum\TwoSum"
3. Enter "dotnet build"
4. Enter "dotnet run"

B)Published application
1. Clone this repository
2. Go to this path "...\week3\Liv\TwoSum\TwoSumPublishedApp"
3. Click the TwoSum Application Manifest toW install the application
