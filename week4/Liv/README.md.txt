### Description
This project is part of my C# week4 training. The app will import data from excel file and perform action to the data.

## How to run the project
A) CLI
1. Clone this repository
2. Open command prompt and go the this path "...\week4\Liv\StudentScore\StudentScore"
3. Enter "dotnet build"
4. Enter "dotnet run"

B)Published application
1. Clone this repository
2. Go to this path "...\week4\Liv\StudentScore\StudentScorePublished"
3. Click the StudentScore Application Manifest to install the application
