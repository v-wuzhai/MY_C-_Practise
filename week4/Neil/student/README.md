# Student

A simple program that reads csv file and display student data.

# Example Usage
```sh
1. cd week4\Neil\student
2. dotnet run student_1000.csv
```

# Note
```sh
1. The program needs to receive argument when running the executable
   (e.g. dotnet run <PATH_TO_CSV_FILE>)
2. the program will ignore the row if both column are string
3. name of student and their score have to be seperated with comma (,)
4. score can be either float or int
```
